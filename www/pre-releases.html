Nextview EPG developers and alpha testers download page

This page lists the latest updates for the nxtvepg software package. The software which is available here is only intended for developers and alpha/beta testers. If you're searching for the regular download page, please visit the homepage.

Downloads

The following packages contain a developer's snapshot of the ongoing development. Please refer to the mailing list archives for information about changes; the changelog in the package may not list all changes or not reflect the actual status of completion.

This release comes with internal consistany checks and debug output enabled and should be used for evaluation only. Please do not redistribute.

  • Win32 executable: nxtvepg-2.5.0-win-dbg.zip
    (identical to the regular release except for additional debugging features)
    (does not contain the Tcl/Tk libraries; please copy them from the last release)

Note: Please use "SHIFT + click" or "save link as..." for .bz2 files (i.e. files compressed with bzip2), as these are not given correct MIME types by the NEFkom server.
