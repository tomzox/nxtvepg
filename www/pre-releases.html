The following package contains a release candidate (rc1) of the upcoming release. For a list of new features see the changelog.

This release comes with debugging enabled and should be used for evaluation only. Please do not redistribute.

Important: rc/INI files written by early pre-releases (2.4.0pre0 - pre2) do not work with this release - please do remove them. Only files written by "official" releases (2.3.1 or earlier) are converted automatically. There's no (incompatible) change between pre3 and pre5.
  • Sources: nxtvepg-2.4.0pre5.tar.gz
  • Win32 binaries: nxtvepg-2.4.0pre5-win.zip

Note: Please use "SHIFT + click" or "save link as..." for .bz2 files (i.e. files compressed with bzip2), as these are not given correct MIME types by the NEFkom server.
