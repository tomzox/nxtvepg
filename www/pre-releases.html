Nextview EPG developers and alpha testers download page

This page lists the latest updates for the nxtvepg software package. The software which is available here is only intended for developers and alpha/beta testers. If you're searching for the regular download page, please visit the homepage.

Downloads

The following packages contain a developer's snapshot of the ongoing development.

Please refer to the mailing list archives and forum for background information.

Changes in pre-releases (note pre4 is the first release candidate for 2.5.2)
  • UNIX: added compiler switch -DZVBI_DECODER in the Makefile to replace VBI (Teletext) slicer with code from libzvbi which performs better on channels with weak reception; sometimes this slicer seems to eat up a lot of CPU (up to 25% on my 400MHz K6-2)
  • UNIX: added support for libzvbi (with or without proxy daemon), enabled by compiler switch -DUSE_LIBZVBI in the Makefile (note this is mutually exclusive with the -DZVBI_DECODER switch)
  • Windows BT8x8 driver: restore settings for all PCI registers upon shutdown (solves problem with Hauppauge WinTV, thanks to Peter Rolf for the info; also solves problem with BtWinCap driver, as pointed out in the discussion forum)
  • Windows: updated & added TV card configuration from DScaler CVS
  • removed German network 3sat from list of known Nextview EPG providers
  • New in 2.5.2pre4:
  • Windows: fixed support for Microtune tuner "MT2032" (e.g. on Pinnacle cards), many thanks to Oliview Avenel for debugging support.
  • Windows: added "Close this menu" entry to systray menu
  • XML export via control menu dialog no longer applies current filter setting
  • fixed compiler warnings for gcc 3.3 (signed/unsigned type clashes)

This release comes with internal consistany checks and debug output enabled and should be used for evaluation only. Please do not redistribute.

  • Sources: nxtvepg-2.5.2pre4.tar.gz
    Only meant for VBI proxy testers (enable -DUSE_LIBZVBI in the nxtvepg Makefile and get the latest version of libzvbi from CVS at zapping.sourceforge.net)
  • Win32 executable: nxtvepg-2.5.2pre4-win.zip
    (does not contain the Tcl/Tk libraries; please copy them from the last release)
  • Teletext/Nextview EPG acquisition (VBI) is still broken in version 0.2.7 of the Linux driver module. Get the latest saa7134 snapshot from bytesex.org
