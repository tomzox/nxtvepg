nxtvepg
Browser für TV-Programmlisten
im XMLTV Format

Herzlich willkommen auf der Homepage von nxtvepg - einer Software zur Darstellung von TV-Programminformationen ("Elektronischer Programmführer" aka EPG) die im XMLTV Format gespeichert sind. nxtvepg ermöglicht ein flexibles Zusammenführen von Programminformationen aus verschiedenen Quellen und deren Anzeige in Form einer Liste oder in Form einer Tabelle ähnlich TV-Programmzeitschriften auf Papier. nxtvepg erlaubt dabei Eingrenzung (aka "Filterung") der angezeigten Programme anhand einer Vielzahl von Kriterien, die Speicherung solcher Eingrenzung als "Shortcuts" und deren Verwendung zur Markierung ausgewählter Programme in der Übersicht mittels selbstdefinierter Spalten. Erinnerungen können für einzelne Sendungen als auch basierend auf Shortcuts programiert werden.

nxtvepg enthält außerdem einen "Teletext EPG Grabber", der mittels einer TV-Karte Programminformationen aus der TV-Programmübersicht im Teletext ausgewählter deutscher TV-Sender extrahieren kann. Die so erzeugten Programmlisten können direkt in nxtvepg angezeigt werden, oder für Verwendung in anderen Programmen in Dateien im XMLTV Format exportiert werden.

Historie: In den Jahren 2000-2008 wurde nxtvepg ursprünlich entwickelt zur Dekodierung und Anzeige von Nextview EPG. Dies war ein Europäischer Standard zur Übertragung von Elektronischen Programmzeitschriften im analogen TV-Signal. Durch den weltweiten Wechsel auf digitale Übertragungsstandards wie DVB ist dieser Standard jedoch obsolet und wurde daher im nxtvepg release 3.0 entfernt. Mehr dazu....

Aktueller release 3.0 (22. März 2021)

Nach einer 10-jährigen Entwicklunspause wurde nxtvepg aktualisiert um zum einen obsolete Features wie "Nextview" und analoge TV-Kartentreiber (Windows) zu entfernen, und im Gegenzug die verbleibenden Features zu optimieren, insbesondere den Import aus XMLTV Dateien und den Teletext EPG Grabber. Details:

  • Der Teletext EPG Grabber unterstützt nun auch den Empfang über digitale TV-Karten (DVB-C/S/T; nur für Linux).
  • Die Geschwindigkeit des Teletext Grabbers wurde erheblich erhöht durch paralleles Laden von mehreren TV-Sendern auf demselben DVB Transponder.
  • Die Verwendung des Teletext EPG Grabbers wurde vereinfacht durch ein neues Kommando im "Control" Menü, das automatisch alle aus Teletext geladenen Programminformationen zusammenführt und anzeigt.
  • Der obsolete Nextview-EPG-spezifische "Serien" Filter wurde durch einen neuen Filter "Wiederkehrende Sendungen" ersetzt.
  • Statistik-Fenster für zusammengeführte Datenbanken und Teletext EPG wurden erweitert gleichwertig zu den früher für Nextview EPG vorhandenen Informationen (u.a. Tortendiagramm und Akquisitionshistorie)
  • Import aus XMLTV wurde vereinfacht zum einen indem nun mehrere Dateien auf der Kommandozeile angegeben werden können, und zum anderen über erweitere Verzeichnisauswahl in den interaktiven Dialogen.
  • Die interne Textkodierung wurde auf UTF-8 umgestellt damit auch Texte von Ländern außerhalb Westeuropas korrekt dargestellt werden.
  • Unterstützung des Nextview EPG Standards wurde komplett entfernt, da dieser Service nicht mehr übertragen wird.
  • Der zuvor integrierte Treiber für analoge TV-Karten auf MS Windows wurde entfernt.

Release 2.9.0 (22. März 2021) - Letzter release mit Nextview EPG

Dieser Release enthält Verbesserungen aus den Jahren 2008 bis 2011, die vormals in Form von "Pre-Releases" 2.9.0pre0-pre4 veröffentlicht wurden. Für den jetzigen Release wurde neben kleinen Korrekturen lediglich eine Aktualisierung vorgenommen die den Software-Build mit aktuellen Tools erlaubt und den Betrieb mit aktuellen Linux Treibern. Der Grund für die Veröffentlichung dieses älteren Softwarestands zur gleichen Zeit wie Release 3.0.0 ist einen letzten Stand mit Nextview EPG zur Verfügung zu stellen, bevor die Unterstützung dieses Standards entfernt wird.

Details zu dem Release finden Sie im Änderungsprotokoll. Diesen Release sollten Sie nur verwenden falls Sie noch Nextview EPG empfangen. Der Release kann von der SourceForge File List heruntergeladen werden.

Feedback

Feedback jeglicher Art ist jederzeit willkommen, sowohl in Form von Fragen, Verbesserungsvorschlägen, Feature requests oder bug reports. nxtvepg hat aktuell sehr wenige Nutzer, daher sollten Sie nicht darauf hoffen dass Bugs behoben werden ohne sie selber zu melden. Folgende Möglichkeiten stehen dafür zur Verfügung:

  • Liste der häufig gestellten Fragen (FAQ) (inklusive Antworten - Stand 2008)
  • Diskussionsforum: die einfachste Möglichkeit (erfordert keine Sourceforge Registrierung). Wenn möglich bitte in Englisch schreiben, oder ansonsten ein [de] dem Titel voranstellen.
