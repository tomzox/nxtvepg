nexTView
EPG Dekoder Software

Die hier zum Download erhältliche Software ermöglicht die Nutzung von nexTView - einer kostenlosen elektronischen Programmzeitschrift die zusammen mit dem Teletext (Videotext) im analogen TV-Signal übertragen wird. Mit dem Dekoder erhalten Sie das Fernsehprogramm aller überregionalen Sender in Deutschland, Österreich, der Schweiz und Frankreich. Einen ersten Eindruck können Sie sich mit diesem Linux screenshot, oder diesem Windows screenshot verschaffen.

Der nexTView Standard wurde von den führenden TV Herstellern unter Führung des European Telecommunications Standards Institute (ETSI) von 1995-1997 entwickelt und ging erstmals zur Internationalen Funkausstellung in Berlin 1997 auf Sendung. Die Absicht war, eine Programmzeitschrift in den Fernseher zu integrieren. Die Übertragung basiert deshalb auf Teletext, jedoch wird ein kompaktes Binärformat anstelle von vorformatierten Textseiten verwendet. Der Vorteil gegenüber den normalen Papier-TV-Zeitschriften ist daß man sofort einen Überbick über alle gerade laufenden Sendungen erhält und zudem die Programmdatenbank nach einer Vielzahl Kriterien durchsuchen kann.

Der nxtvepg Dekoder erlaubt es nun, nexTView auch am PC zu nutzen und bietet gleichzeitig eine Menge neuer Features, die in einem TV-Gerät nicht realisierbar wären. Die einzigen Vorbedingungen für Sie als Nutzer sind daß Sie (a) eine TV-Tuner-Karte im PC stecken haben und (b) damit einen der Sender empfangen können auf dem nexTView übertragen wird.

Momentan wird nexTView auf den folgenden Sendern ausgestrahlt:

  • in Deutschland und Österreich: Pro7, 3Sat, RTL-II.
  • in der Schweiz: u.a. auf EuroNews und TSR1.
  • in Frankreich: Canal+, M6, TV5.

    • Nährere Details finden sich in der Anbietertabelle. Sollten Sie keinen der genannten Sender empfangen, können Sie mit dieser Software mit Ausnahme des Demo-Modus leider nichts anfangen. Mehr Infos dazu und zu allen weiteren Sofware-Features in der nxtvepg Dokumentation (nur in englisch verfügbar).

    Falls nxtvepg Ihr Interesse geweckt haben sollte und Sie nun an einem TV mit integriertem nexTView interessiert sind, schauen Sie doch auf den Seiten der Hersteller vorbei: Grundig, Loewe, Metz, Philips, Sony or Thomson.

    Systemanforderungen

    Harwarevoraussetzungen sind ein Pentium PC und eine TV-Karte mit dem Booktree Chip Bt848 oder Bt878.

    Linux Softwareanforderungen

  • kernel >= 2.0.35 (upgrade auf 2.2.x für bttv-Nutzer wärmstens empfohlen)
  • bttv Treiber (version >= 0.7.35 empfohlen) und video4linux
  • GNU C Compiler (falls Sie nxtvepg nicht in Form eines RPM Binärarchivs herunterladen)
  • Tcl/Tk Version 8.0 oder neuere (8.3 empfohlen) verfügbar unter ftp://www.scriptics.com/pub/tcl/

    • NetBSD Softwareanforderungen

  • Kernel >= 1.5 mit dem bktr Treiber Version 2.17 oder neuer,
  • GNU C Compiler (falls Sie nxtvepg nicht in Form eines vorkompilierten NetBSD packages herunterladen)
  • Tcl/Tk Version 8.0 oder neuere (8.3 empfohlen) verfügbar unter ftp://www.scriptics.com/pub/tcl/

    • Windows Softwareanforderungen

  • Windows NT oder Windows 95. Andere Windows-Varianten sind nicht getestet, dürften aber keine Probleme bereiten.
  • Auf Windows-NT sind Administrator-Rechte erforderlich (um den Treiber für die TV-Tuner-Karte beim Programmstart zu laden)
  • Treiber für Bt8x8: Download-Anweisungen finden sich im README.txt im ZIP-File.

    • Download

    Die neueste Distribution 0.5.1 (Unix) bzw. 1.7-win (Windows) vom 5. Juni 2001 schließt den Kreis in der Anbindung an xawtv (UNIX TV Programm): Nun kann nicht nur das TV-Programm direkt aus der Programmübersicht heraus umgeschalten werden, sondern nxtvepg folgt auch Programmwechseln durch xawtv mit dem Cursor in der Übersicht. Der Titel der gerade laufenden Sendung wird außerdem in einem kleinen Popup unterhalb xawtv eingeblendet.

    Der bugfix release 0.5.2 bzw. 1.8-win behebt zwei Fehler in optionalen Features. Eine Zusammenfassung aller seit Version 0.5.0 hinzugekommenen Features und bugfixes befindet sich im Change-Log. Falls mit der Software Probleme auftauchen, bitte melden. Ältere Distributionen befinden sich noch im Source Verzeichnis (US Mirror)

    Die Windows Version wurde auf einen neuen Compiler umgestellt (gcc-mingw32). Falls Probleme auftauchen, bitte melden.

    nxtvepg-1.8-win.zip (902 kB)

    Bitte beachten daß ein weiterer Treiber benötigt wird, der nicht in diesem Paket enthalten ist und separat heruntergeladen werden muß. In manchen Fällen kann auch der Treiber anderer TV-Applikationen verwendet werden. Die nötigen Infos dazu finden sich im README.txt im ZIP file.

    Demo Datenbank

    Falls Sie keine der oben genannten nexTView Programmanbieter empfangen, können Sie die Software dennoch im Demo-Modus ausprobieren. Dazu können Sie diese Demo Datenbank (57 kB) downloaden, dekomprimieren und dann mit den Kommandozeilenargumenten -demo nxdemodb.epg laden oder im Windows-Explorer einfach per Drag-and-Drop auf das nxtvepg Executable fallen lassen.

    Das Urheberrecht am Inhalt dieser Datenbank liegt bei Swiss Tele text, deshalb bitte nicht weiterverbreiten.
