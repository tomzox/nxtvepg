nexTView
EPG Dekoder Software

Herzlich willkommen auf der Homepage von nxtvepg - einer Software die es ermöglicht Nextview EPG am PC zu nutzen. Nextview ist ein ETSI-Standard zur Übertragung von elektronischen Programmzeitschriften im TV-Signal (egal ob analog oder digital, d.h. DVB). Dieser kostenlose Service wird derzeit von verschiedenen Anbietern in Deutschland, Österreich, der Schweiz, Frankreich und Belgien ausgestrahlt. Zusammengnommen decken sie das tägliche Fernsehprogramm aller überregionalen Sender in diesen Ländern ab. Weiter...

Current releases 2.7.7 and 2.8.0 (Dec/31/2007)

Nach 3 Jahren Entwicklungszeit gibt es nun endlich einen offiziellen Release 2.8.0 mit einer Reihe größerer neuer Features. Der vorhergehende Release 2.7.7 enthält aber auch (fast) alle wichtigen bug-fixes und bleibt somit als Alternative falls mit dem neuen Release Probleme auftreten.

Die größten Neuerungen im Release 2.8.0 sind:

  • Abspaltung eines reinen EPG Akquisitions-Hintergrundprozesses ohne GUI ("nxtvepgd")
    (Auf vielfachen Wunsch der Anwender mit Linux-basierten HTPCs ohne X11 Libraries.)
  • Abgreifen von EPG Daten aus regulären Teletext-Seiten deutscher Sender (Mehr Info...)
    (Dies erfordert die Installation eines Perl Interpreters.
    Das Feature ist für UNIX auch als separates Perl Skript verfügbar.)
  • Import und Zusammenführung von EPG Daten aus externen Dateien im XMLTV Format.
  • Erweitertes Kommunikationsprotokoll für OSD in TV-Applicationen (für UNIX und Windows)
    (Wichtig: Aufgrund dieser Änderung müssen alle Anwender von K!TV ihr Nextview EPG Plugin austauschen.)
  • Verbesserte Unterstützung für "exotische" TV-Karten unter MS Windows.
    (Dies erfordert allerdings eine separate WDM-Treiber-"Schnittstellenbibliothek")
    Vielen Dank an Gérard Chevalier für die Implementierung der Bibliothek.

Zum Herunterladen in der Download Sektion.
Siehe auch die detaillierte Liste aller Änderungen im Change Log.

Feedback

Feedback jeglicher Art ist jederzeit willkommen, sowohl in Form von Fragen, Verbesserungsvorschlägen, Feature requests oder bug reports. Folgende Möglichkeiten stehen dafür zur Verfügung:

  • Liste der häufig gestellten Fragen (FAQ) (inklusive Antworten)
  • Diskussionsforum: die einfachste Möglichkeit (erfordert keine Sourceforge Registrierung). Wenn möglich bitte in Englisch schreiben, oder ansonsten ein [de] dem Titel voranstellen.
  • Mailing list: Auf dieser Liste werden hauptsächlich Software-Releases angekündigt, einschließlich Testversionen (Alpha- und Beta Releases) sie steht aber auch zur Diskussion zwischen Entwickler(n) und Anwendern zur Verfügung. Die Listensprache ist Englisch, Anmeldung erforderlich.
  • Um über neue releases auf dem Laufenden zu bleiben, kann das nxtvepg Projekt auf freshmeat.net abonniert werden.

Bevor Sie eine Frage stellen überlegen Sie bitte ob sie durch das manual geklärt wird und sehen Sie im mailing list Archiv als auch im Forum nach ob dieselbe Frage eventuell bereits kürzlich beantwortet wurde.
