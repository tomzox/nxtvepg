Linux Nextview EPG Decoder Download Page

In this software package you find a decoder for Nextview - an Electronic Programme Guide for the analog domain (as opposed to the various digital EPGs that come with most digital broadcasts). It allows you to receive and browse free TV programme listings for most of the major networks in Germany, Austria, France and Switzerland. For detailed information please refer to the manual page or this screenshot.

Currently Nextview EPG is transmitted by the following TV stations:

  • in Germany and Austria: Pro7, 3Sat, RTL-II.
  • in Switzerland: EuroNews, TSR1 and others.
  • in France: Canal+, M6, TV5.

    • If you don't receive any of these, then this software is useless to you (except for a demo mode - see below).

    The Nextview standard was developed by the major European consumer electronics manufacturers under the hood of the European Telecommunications Standards Institute (ETSI) in 1995-1997. The author of this site's software has no connections whatsoever to the ETSI - but he still hopes it's kindly tolerated.

    System Requirements

    To compile and use this software, you will need:
  • Pentium class PC with a Bt848 or Bt878 based TV tuner card
  • Linux with kernel >= 2.0.35 and bttv driver and video4linux
  • GNU C Compiler
  • Tcl/Tk version 8.0 or later (ftp://www.scriptics.com/pub/tcl/)

    • As of now, there's no supprt for Micro$oft Windows. Please read the file windows

    Download

  • Download the latest source distribution: nxtvepg-0.3.4.tar.gz (160 kB)

  • In case of trouble, download the previous distribution: nxtvepg-0.3.3.tar.gz (150 kB)

  • If you do not receive any of the Nextview providers listed above, download the demo database nxtvdb-tv5.gz and load it with the command line arguments -demo nxtvdb-tv5 into nxtvepg (the copyright of this data is by Swiss Tele+text - please do not redistribute).
