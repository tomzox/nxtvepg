nexTView
EPG Decoder Software

In this software package you find a decoder for nexTView - an Electronic TV Programme Guide for the analog domain (as opposed to the various digital EPGs that come with most digital broadcasts). It enables you to receive and browse free TV programme listings for all of the major networks in Germany, Austria, France and Switzerland. You can have a look at it in this Linux screenshot (new for version 2.4.0), or this Windows screenshot.

The nexTView standard was developed by the major European consumer electronics manufacturers under the hood of the European Telecommunications Standards Institute (ETSI) in 1995-1997 with the aim to offer an "electronic TV magazine" inside their analog TV sets. The transmission protocol is based on teletext, however using a compact binary format instead of preformatted text pages. The advantage compared to paper-based TV magazines is that you have an immediate overview what's running Now & Next, plus you can search through the programme database by many different filter categories.

Now the nxtvepg software enables you to use nexTView with your PC too, and adds a lot of features that would not be possible to implement inside a TV set. The only preconditions are (a) that you have a TV tuner card for TV reception inside your PC and (b) that you receive one of the nexTView content provider's TV channels.

Currently nexTView EPG is transmitted by the following TV stations:

  • in Germany and Austria: Kabel1, 3Sat, RTL-II, EuroNews (coverage: apx. 31 networks)
  • in Switzerland: SF1, TSR1, TSI1, EuroNews, 3sat, Kabel1 (coverage: apx. 37 networks)
  • in France: Canal+, M6 (coverage: 8 networks)
  • in Turkey: TRT-1 (coverage: 17 networks)

For more details see the provider table. If you don't receive any of these, then unfortunately this software is useless to you, except for a demo mode. For detailed information please refer to the nxtvepg manual page.

If you're interested in buying a TV with nexTView inside, look for it at one of the following manufacturers: Grundig, Loewe, Metz, Philips, Sony, Thomson or Quelle Universum.

System Requirements

Hardware requirements are a PC or workstation with a 90 MHz Intel Pentium or equivalently (or more) powerful CPU and a TV tuner card based on the Booktree chip Bt848 or Bt878. (Almost all TV cards for PC except the ATI "TV Wonder" are built around one of the Booktree chips.)

Linux software requirements

  • kernel >= 2.0.35 (upgrade to 2.2.x highly recommended for bttv users)
  • bttv driver (version >= 0.7.35 recommended) and video4linux
  • GNU C Compiler
  • Tcl/Tk version 8.0 or later (8.3 recommended) available from ftp://www.scriptics.com/pub/tcl/

NetBSD software requirements

  • kernel >= 1.5 including the bktr driver 2.17 or later,
  • GNU C Compiler, unless you got nxtvepg in a precompiled NetBSD package
  • Tcl/Tk version 8.0 or later (8.3 recommended) available from ftp://www.scriptics.com/pub/tcl/

Windows software requirements

  • New since 2.0.0: No additional software is required: nxtvepg is shipped with all the required components.
  • Exception for Windows 95: the winsock2 DLL is required (ws2_32.dll, for the daemon feature) which - in contrary to newer Windows versions - was not included with this OS.
  • New since 2.0.0: All Windows versions starting with Windows 95 are supported.
  • Note: On Windows NT and Windows 2000 you need to run nxtvepg under an account with special privileges (you must be allowed to install & start a service)

Download

The latest release offers major improvements, including user defined programme display (e.g. print titles of your favorite series in bold red text), inverting searches (e.g. show all programmes that do NOT have adult content), export in XML format, and many more (details see change log 2.4.0).

Source distribution

Download the latest stable source distribution for Linux and NetBSD: nxtvepg-2.3.1.tar.gz

Download the newest (possibly unstable) source distribution: nxtvepg-2.4.0.tar.gz

You can build a RPM package from the source tarball simply by running rpm -ta nxtvepg-2.4.0.tar.gz and install it as usual with rpm -i. You can also donwload directly installable packages (not necessarily the latest version) for Linux at rpmfind.net or Debian and for NetBSD at netbsd.org

Binary distribution for M$ Windows

Windows executable of the latest stable version: nxtvepg-2.3.1-win.zip

Windows executable of the newest (possibly unstable) version: nxtvepg-2.4.0-win.zip

Demo application for TV application interaction: nxtvepg-2.4.0-tvsim.zip
(see tvsim, vbirec or vbiplay manuals, or tvsim & nxtvepg and tvsim & vbirec screenshot)

Simply unpack the nxtvepg ZIP file into an empty directory and then double-click the nxtvepg.exe file. If you're upgrading from an older version of nxtvepg, you can copy your previous INI file and databases into the new directory. The optional tvsim ZIP file should be unpacked into the same directory. Please refer to the Help menu or HTML manual page, section Getting Started for setup instructions.

Demo Database

If you do not receive any of the nexTView provider channels listed at the top of this page, download the demo database nxdemodb.epg.gz, which will enables you to check out the main features of the browser, so that you can see what you're missing.

To use the demo database, download it into your nxtvepg directory, decompress it and load it with the command line arguments -demo nxdemodb.epg (Unix) into nxtvepg, or simply drag and drop it onto the executable (Windows Explorer).
