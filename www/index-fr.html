nexTView
Logiciel de Décodage EPG

Bienvenue sur la page d'accueil du progiciel de nxtvepg : Un décodeur pour nexTView EPG, un guide électronique de programmes TV dans le domaine analogique (par opposition au EPGs numériques qui viennent avec les émissions numériques, bien que vous puissiez recevoir Nextview EPG numérique, aussi). Il vous permet de recevoir et passer en revue les programmes TV pour tous les réseaux principaux en Allemagne, Autriche, la Suisse, la France et la Belgique par l'intermédiaire d'une carte de TV analogique, sans accès à Internet. Version 2.7.4 (27.12.2004)

Release 2.7.4 once more is mainly a maintenance release with mostly bug fixes and a few feature enhancements. Highlights include:

  • Fixed a bug in EPG teletext number updates which caused acquisition to stall if multiple providers were acquired which used different teletext pages for transmission (e.g. in Switzerland)
  • Fixed timezone calculation for provider "VT4" in Belgium
  • Fixed bug in text search filter for merged databases: sub-string search ended after 258 characters
  • Added possibility to store "expire time" filter in shortcuts
  • Added possibility to configure text foreground colors (at least for programme list) to support an inverted color scheme (configurable via .Xresources or Nxtvepg.ad)
  • Added new display format option for composite attributes (i.e. user-defined columns): set foreground color for entire column

Regardez aussi la liste détaillée des modifications majeures.

Questions et feedback

Vos feedback sont toujours les bienvenus. Si vous avez des questions spécifiques, des suggestions, un rapport de bogue ou n'importe quel autre genre de feedback, libre à vous d'employer une des possibilités suivantes:

  • Frequently Asked Questions (FAQ)
  • Forum de discussion: Endroit préféré pour des questions, des demandes de dispositifs, des rapports de bogues ou l'échange de trucs et astuces entre les utilisateurs (n'exige pas l'enregistrement à Sourceforge).
  • Mailing list: Actuellement utilisée seulement pour des annonces de pré-release. Mais elle est également ouverte pour des discussions techniques entre les développeurs et les utilisateurs du logiciel de nxtvepg. Inscription exigée.
  • Si vous voulez seulement être averti de la sortie officielle de releases, souscrivez au projet nxtvepg sur freshmeat.net.

Avant de poster une question, vérifiez svp si la réponse ne se trouve pas dans le manuel et regardez dans les archives de la mailing list et le forum si la même question n'a pas été posée récemment.

Recherche aide

Si vous êtes un développeur de logiciel et que vous aimez nxtvepg et que vous voulez le rendre bien plus utile et plus pratique ou si vous cherchez un projet freeware pour travailler dessus, svp apportez votre contribution à nxtvepg. J'ai compilé une liste de sujets qui bénéficieraient de votre aide. Notez que vous pouvez également contribuer en ajoutant le support de nxtvepg à d'autres applications, par exemple en ajoutant Nextview EPG on-screen-display (osd) dans différentes applications de visionnement de TV.
